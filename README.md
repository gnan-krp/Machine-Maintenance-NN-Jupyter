# Machine Maintenance Prediction Project
# =====================================
 
# Jupyter Notebook Implementation
 -------------------------------
# Features:
 - All steps in a single notebook: data, preprocessing, model, training, plots, predictions
 - Graphs displayed inline in the notebook
 - Optional saving of graphs for GitHub or reports
# 
# Run using:
   jupyter notebook machine_maintenance.ipynb
# 
# Requirements:
 -------------
# Python >= 3.8
# 
# Packages:
 - numpy
 - pandas
 - matplotlib
 - scikit-learn
 - tensorflow
# 
# Install packages:
 ----------------
 pip install -r requirements.txt
# 
# How to Run:
 -----------
# Python Scripts:
# 1. Make sure your CSV dataset is in the data/ folder.
# 2. Run the main script:
    python main.py
    - Training will start
    - Graphs will be saved in visuals/
    - Predictions printed in console
# 
# Jupyter Notebook:
# 1. Open the notebook:
    jupyter notebook machine_maintenance.ipynb
# 2. Run cells sequentially to see inline graphs and predictions.
# 
# Outputs:
 --------
 - Training Loss Curve – Training vs Validation Loss
 - Prediction vs Actual – Scatter plot comparing predicted and actual maintenance costs
 - Single-sample prediction – Predicted cost for a specific machine
